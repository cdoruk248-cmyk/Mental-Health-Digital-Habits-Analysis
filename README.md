# Multi-Dataset Mental Health & Digital Habits Analysis 🧠🎮💼

This project analyzes the intersections between digital habits (gaming addiction), professional workplace environments (tech industry burnout), and general mental health indicators using three distinct datasets (Total N = 5750).

## 🚀 Key Features & Methodology
- **Data Preprocessing & Cleaning:** Handled missing values and standardized psychological metrics.
- **Exploratory Data Analysis (EDA):** Visualized key correlations between daily screen time, anxiety, stress, and sleep deprivation.
- **Advanced Machine Learning (K-Means Clustering):** Segmented gamers into 3 distinct risk profiles (Balanced, Escapist, Chronic Addict) based on playtime, stress, and addiction scores.
- **Comparative Industry Analysis:** Evaluated the impact of remote vs. in-person work arrangements on tech employee burnout.

## 📊 Key Insights
1. **The Gaming-Stress Paradox:** The highest stress levels (7.61/10) were found in the "Escapist" group (5 hrs/day) rather than the most extreme gamers, showing that gaming is heavily used as a coping mechanism for underlying stress.
2. **Physical Toll:** "Chronic Addict" gamers (8.4+ hours/day) experience a severe drop in sleep quality, averaging just 6.2 hours per night.
3. **Remote Work Beats Burnout:** Remote tech employees reported the lowest burnout scores (5.63), while in-person workers suffered the highest (6.03), debunking traditional office productivity myths.
